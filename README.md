## Clone project
```shell
git clone https://github.com/Ssspook/CHIMP.git
```

## Dependencies

```shell
make install
```

## Run

```shell
make start-frontend
make start-backend
```

Full commands list:

```shell
make help
```

| Address                                   | Description    |
|-------------------------------------------|----------------|
| http://localhost:5173/                    | Frontend    |
| https://localhost:7186/api/               | API         |
| https://localhost:7186/swagger/index.html | Swagger     |
